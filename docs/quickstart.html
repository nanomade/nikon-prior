Nikon-Prior — Quick Start & Reference

Allied Vision Alvium 1800 U-508c  |  Prior ProScan III  |  Nikon L200ND  —  v2025

First-Time / Startup

Launch

  1. Activate the Python venv: source ~/nikon-prior/bin/activate
  2. Run: QT_QPA_PLATFORM=xcb python main.py
  3. The Launcher window opens. Click buttons to open panels.
  4. Click Open Camera Preview → live image appears.
  5. Click Open Imaging Controls → adjust exposure, binning, WB.
No camera? App falls back to MockCamera (synthetic noise). Check USB connection and Vimba X runtime if camera is missing.
"Camera already in use" — run pkill -f main.py or replug USB.

Typical imaging workflow

  • Select magnification in Imaging Controls
  • Set binning to 4x for fast overview, 1x for full resolution
  • Adjust exposure (log slider, 0.01–1000 ms) or enable Auto Exposure
  • Set white balance temperature (~5300 K for daylight LEDs)
  • Navigate stage with XY sliders or joystick
  • Focus with Z slider or run Autofocus
  • Enable Scale Bar; disable when saving for analysis
  • Use Capture Frame (raw) or Save View (screen)

Autofocus Reference

Running autofocus

  1. Open Focus Panel from Launcher
  2. Set Search range (e.g. 200 µm) and Step size (e.g. 5 µm)
  3. Click Auto Focus — stage sweeps Z, finds peak sharpness
  4. Stage moves to best-focus Z position automatically
Uses Laplacian variance metric, normalised to 616×514 px so settings are consistent across all binning modes.

Sample Tools Reference

Find Wafer Extents

  1. Place wafer on stage (rough centre)
  2. Open Edge Detection Panel
  3. Set threshold and direction, click Find Edge
  4. Repeat for all four edges
  5. Click Go to Centre — stage moves to wafer centre

Wafer Scan

  • Define scan grid (rows × columns, step size)
  • Click Start Scan — stage snake-scans the area
  • Each position auto-focuses (optional) and saves an image
  • Images saved with XY coordinates in filename
Use 4x binning for fast overview scans; switch to 1x for high-res acquisition.

Index Mark Navigator

  • Teach the index mark position once (“Set Reference”)
  • Navigate to any saved site relative to the index mark
  • Compensates for wafer placement variation between sessions

Quick Troubleshooting

Camera issues

SymptomFix
Mock camera / no live imageCheck USB3 cable; check Vimba X runtime installed; pkill -f main.py then restart
Image very darkIncrease exposure; check illumination is on; try Auto Exposure briefly
Image washed out / saturatedReduce exposure; disable Auto Exposure
Colour castAdjust WB temperature slider
Flickering imageSet Flicker Averaging to 3–4 frames

Stage issues

SymptomFix
Stage not respondingCheck USB-serial cable; verify /dev/ttyACM0 present; restart app
Sliders out of sync after joystick useNormal — sliders resync on next startup
Stage moves wrong directionCheck invert flags in motors/step_config.json
Z drifts / overshootReduce Z step size; check mechanical coupling of H122KON kit
